# Nombres_aleatoires

Générateur de nombres aléatoires
Travail en fortran 95 au format libre

La compilation se fera alors avec 

>$ gfortran -o rnd rnd.f95

pour exécuter le programme dans le terminal avec

>$ ./rnd.exe
